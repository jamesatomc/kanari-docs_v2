# Kanari Network

Kanari Network is a secure blockchain platform designed for **file metadata storage and verification**. Powered by the **Move Virtual Machine (Move VM)**, it offers robust smart contract capabilities and transparent metadata management in the Web3 ecosystem.

## Key Features

- **Secure Metadata Storage**: Anchor file metadata on-chain for reliability.
- **Move VM Integration**: Leverage secure and verifiable smart contracts.
- **Ownership Verification**: Track file ownership and history immutably.
- **Web3 Ready**: Seamlessly bridge file storage solutions like IPFS with on-chain metadata.

## Why Kanari Network?

While solutions like IPFS handle P2P file storage, Kanari Network focuses on anchoring metadata on-chain, ensuring:
- **Data Integrity**: Proof of file authenticity.
- **Transparent Versioning**: Track changes and history.
- **Secure Access Control**: Permissionless and trustless operation.
